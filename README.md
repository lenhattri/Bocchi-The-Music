## Giới thiệu
- Đây là dự án môn Lập Trình Web

## Dependencies
- MySQL
- Composer
- PHP
## Cách cài đặt

```
cp .env.example .env
composer install
php artisan migrate
```
 - Cài npm
```
npm install
```

- Phân ra làm 2 terminal:

    + 1 cái chạy:

           
            npm run dev
      
      
    + 1 cái chạy:

      
            php artisan serve
